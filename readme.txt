== Installation ==

Step-by-step Installation Guide:

1. Log into your Wordpress panel and click 'Add new' in the 'Plugins' menu.
2. Install Chatwee Live Chat plugin from WordPress directory and activate it.
3. Sign in to your Chatwee control panel on http://chatwee.com. Sign Up Free if you don't have account yet.
4. Copy the code snippet that appears on the script page.
5. Paste it in right place to Chatwee WordPress plugin settings page, and click 'Save Changes' button.
6. That's it! Always return to Chatwee control panel to customize dozens of settings.
